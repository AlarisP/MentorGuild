# main.Java.Com.MentorGuild.main.Java.Com.main.Java.Com.MentorGuild.main.Java.Com.MentorGuild
A fun, fan-driven learning platform where famous personalities teach unexpected subjects through chaotic humor and pop-culture analogies, starting with K-pop idols explaining cybersecurity concepts.


Test Java on a local host:
1. download JDK and Maven (I used chocolaty to help )
2. run
   mvn clean install
3. run
   mvn spring-boot:run
4. Go to browser and type
   http://localhost:8080/lessons/
